Overview

WeDo is a mobile app designed to make wedding planning easier and more affordable. This project was completed as part of the Google UX Design Certificate program, and it entailed end-to-end UX design skills from initial research through final prototyping in Figma.

Key Takeaways

This project helped me learn through hands-on experience, through user research, wire framing, and usability testing. I loved each stage of the process, from sketching wireframes on a notepad (see slideshow above) and evolving the design based on user feedback. Creating a comprehensive design system from scratch was especially rewarding.
